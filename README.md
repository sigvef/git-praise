# git praise

A nicer `git blame`.

## Installation

```
pip install git-praise
```

Make sure that you have your python package bin path in your $PATH.
On Ubuntu, this is `~/.local/bin`.

## Usage

```
$ praise path/to/my/file
```

## Development setup

```
git clone git@github.com:sigvef/git-praise.git
cd git-praise
make setup
```
